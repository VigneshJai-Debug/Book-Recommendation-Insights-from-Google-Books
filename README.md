# Book-Recommendation-Insights-from-Google-Books
Analysis of Book Metadata and Popularity Trends Using Google Books API

The dataset consists of metadata for books retrieved from the Google Books API. It includes fields such as book title, author(s), publisher, publication year, categories (genres), page count, language, average rating, and total number of ratings. The dataset is collected in JSON format and later converted to CSV for analysis. It is suitable for analyzing book popularity, genre trends, and recommendation systems.


SOURCE OF DATASET: Dataset collected programmatically using the Google Books Public REST API. The API provides structured JSON data containing book metadata including title, authors, genres, ratings, and publication information.
